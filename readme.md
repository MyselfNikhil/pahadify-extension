# Pahadify - Image converter Extension

A lightning-fast, privacy-first browser extension for local image conversion and processing. This repository contains the extension architecture tied to **[Pahadify - Free Serverless Image Editor](https://thepahadify.com)**.

## Overview
This utility is built on the philosophy of zero-server processing. It allows users to manage and convert image formats locally without ever uploading files to a remote cloud server. 

### Key Features:
* **Instant WebP Conversion:** Right-click to convert heavy WebP files into standard JPG or PNG formats directly within the browser context.
* **100% Client-Side:** Powered entirely by local HTML, CSS, and JavaScript. Your images never leave your machine.
* **Privacy-Centric:** No tracking, no uploads, no server storage.

## Integration
This extension is a standalone utility that complements the main web application. For advanced features including smart compression, multi-layer management, AI image generation, and 20+ different manipulation tools, visit the full Progressive Web App.
